OAuth 2.0 On-Behalf-Of Flow Demo

Welcome to the OAuth 2.0 On-Behalf-Of (OBO) Flow demonstration!

This application demonstrates secure service-to-service communication where:

  • You log in through Keycloak
  • The Client App calls API Service 1 with your token
  • API Service 1 exchanges your token for a new token to call API Service 2
  • API Service 2 validates the token and returns secure data
